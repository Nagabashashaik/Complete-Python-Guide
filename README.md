# Nagabasha’s Ultimate Python Crash Course

## PHASE 0: History of Python

### What is Python?
- **Python** is a popular, high-level programming language, which is widely used for web and software development, task automation, data analysis, and artificial intelligence.
- It is known for its simplicity and readability.

---

### How was the world before Python?
- In the late 1980s, if you want to write a program, you generally had to pick a language from **C**, **C++**, **Bash/Shell scripting**, **ABC**, or **Perl**.
- But there are some limitations in the late 80’s:
  - **C/C++** – powerful but complex, hard to learn, need manual memory management
  - **Perl** – good for text tasks, but messy and hard to read
  - **Shell scripts** – useful for small tasks, not for large programs
  - **ABC** – simple and readable, but too limited for real-world use
- **Problem:** You either got *power* (C/C++) or *simplicity* (ABC) — but never both.

---

### Who created Python and how it was created?
- Created by **Guido van Rossum** in **1989**, released in **1991**
- He worked on **ABC** and liked its simplicity, but wanted something more powerful and flexible
- So, he started creating Python during the Christmas holidays
- He wanted a **short, unique, and mysterious name**, so he named **PYTHON** by taking inspiration from *Monty Python's Flying Circus* (British sketch comedy show)
- **Solution:** This bridges the gap between simplicity and power — both in a single language

---

### How was the world after Python?
- **1990’s** – adopting stage: used for scripting, sysadmins.
- **2000’s** – it has grew and used in YouTube, Instagram, Reddit, Dropbox, mainly Django available for web development
- **2010’s** – exploded with **data science** and **AI/ML**
- **2020’s to now** – became the **#1 language for AI**, most-taught beginner language, used everywhere from finance to NASA

---

### Features of Python
1. **Easy to Learn & Read** - Contains simple, English-like syntax
2. **Free and Open Source** - Completely free to use
3. **Portable** - Same code runs on different operating systems (Windows, Linux, macOS) without changes
4. **Interpreted Language** - Code runs line-by-line, easy to find errors and fix them
5. **Massive ecosystem** - Gives access to hundreds of thousands of libraries; no need to write everything from scratch
6. **Object-Oriented** - Supports classes, objects, inheritance, encapsulation
7. **Extensible & Embedded** - Can integrate with C/C++ code; can be embedded into other applications as a scripting layer
8. **Strong Community Support** - One of the largest developer communities in the world

### Limitations of Python
- **Slow execution** – Interpreted language, so it runs slower than C, C++, or Java.
- **High memory usage** – Uses more memory due to flexibility and dynamic typing.
- **GIL limitation** – Only one thread runs at a time, limiting true multi-threading.
- **Weak mobile development** – Not commonly used for native iOS/Android apps.
- **Runtime errors** – Type-related bugs often appear only when the code runs.
- **No browser support** – Can't run natively inside web browsers like JavaScript.
- **Dependency management issues** – Package/version conflicts can get messy.
- **Not ideal for real-time systems** – Rarely used for OS-level or hardware-level programming.
- **Less mature database layer** – Database tools are less polished than languages like Java.
